oUF_Phanx
============

An oUF layout optimized for PvE, with many features and some options.

It includes frames for the player, pet, target, targettarget, focus,
focustarget, party, party pet, arena, arena pet, and boss units.

It *does not* include frames for raid or raid pet units. Use Grid or
another raid frame addon of your choice.

Requires [oUF] [1] 1.6 or higher.

	[1]: http://www.wowinterface.com/downloads/info9994-oUF.html


Download
-----------

* [WoWInterface](http://www.wowinterface.com/downloads/info13993-oUF_Phanx.html)
* [Curse](http://wow.curseforge.com/addons/ouf-phanx/)


Features
-----------

* Aggro highlighting
* Buff and debuff filtering, with in-game options
* Hold Shift out of combat to temporarily disable buff filtering
* Debuff highlighting, with defensive and offensive dispel priorities
* Incoming heals overlaid on health bars
* Bars for druid mana and eclipse energy, runes, and totems
* Combo point-style display for Arcane Charges, Burning Embers, Chi, 
  Demonic Fury, Holy Power, Maelstom Weapon, Shadow Orbs, Soul Shards,
  Wild Mushrooms, and (of course) combo points
* Casting bars on player, pet, target, and focus frames
* Resurrection and Soulstone status text
* AFK timers on player and party frames
* Optional combat feedback text
* More detailed health and power text on mouseover
* Automatic adjustment of some elements based on role


### Supported Plugins

* oUF_MovableFrames
* oUF_SmoothUpdate
* oUF_SpellRange


Options
----------

Type “/pouf” for options, or find them in the Interface Options window.

If you want to change something that doesn’t have an option to change,
you will need to modify the layout’s code yourself, and I will *not*
provide any support or assistance for this. If you need help modifying
the code, post a thread in the [oUF forum on WoWInterface] [2].

	[2]: http://www.wowinterface.com/forums/forumdisplay.php?f=87


Localization
---------------

Compatible with all localized game clients.

Translated into English, Deutsch, Español, Français (partial),
Português (partial), Русский (partial) and 简体中文 (partial).

Add or update translations on the [CurseForge project page] [3].

	[3]: http://wow.curseforge.com/addons/ouf-phanx/localization/


Feedback
-----------

Post a ticket on either download site, or a comment on WoWInterface.

If you are reporting a bug, please include directions I can follow to
reproduce the bug, whether it still happens when all other addons are
disabled, and the exact text of the related error message (if any) from 
[BugSack](http://www.wowinterface.com/downloads/info5995-BugSack.html).

If you need to contact me privately, you can send me a private message
on either download site, or email me at <addons@phanx.net>.
